#  壹無前端

idea：壹無（易物），barter。

风格：繁体古风。

logo：双手相扣。

用户身份：普通用户、管理员。

功能：以物易物的论坛。看帖（评论、点赞）、发帖、我的。

模块：看、写（删）、我（与我相关帖）。

页面设计：主要页面设计为top+button+content。

价值：闲置物品重新利用，提高资源利用率，响应供给侧改革。

技术方案：`springboot+vue`。
